## 🛠️ 사용한 기술 (Tech Stack)

- **Frontend:** Next.js 15+, React 19, Tailwind CSS 4
- **Infrastructure:** AWS S3 + CloudFront
- **DNS & Security:** Cloudflare + AWS Certificate Manager

---

## 🚀 배포 방법 (나를 위한 메모)

수정한 내용을 사이트에 반영하고 싶을 땐 터미널에 딱 두 줄만 입력해요.

1.  **빌드하기:** `yarn build`
2.  **AWS에 올리기:** `aws s3 sync out/ s3://YOUR_BUCKET_NAME --delete`
